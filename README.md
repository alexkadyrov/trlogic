
<p>
Простое REST API с одним единственным методом, который загружает изображения.
</p>

<p>
Реализовано на go 1.11
<br>
В качестве менеджера зависимостей используется go modules (разворачивать за пределами GOPATH)
</p>

<p>
Запуск:
<br>
go run api/main.go
<br>
или
<br>
docker-compose up
</p>
